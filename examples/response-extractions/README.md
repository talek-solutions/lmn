# Response Extraction Examples

Used with `-S` / `--response-template`. Each file defines JSON paths to extract from response bodies for tracking in the results summary.

| File | Description |
|------|-------------|
| `nested-string.json` | Extracts a nested string field (e.g. an error code) from the response body |

See [TEMPLATES.md](../../lmn-core/TEMPLATES.md) for the response template schema.
